### 注意
```
 由于cnpm 存在丢包现象，所以请使用 yarn 包管理工具  （cnpm install yarn -g）
```
### 开发
*  yarn dll -- -p   # 首次或dll变动，编译（仅一次）
*  yarn dev

```
启动后浏览器访问[http://localhost:3000](http://localhost:3000)
```

### 测试
 * yarn dll -- -p   # 首次或dll变动，编译（仅一次）
 * yarn test

```
 将项目打包到 `build` 目录下
```

### 部署生产
  * yarn dll -- -p   # 首次或dll变动，编译（仅一次）
  * yarn build

```
 将项目打包到 `build` 目录下
```
